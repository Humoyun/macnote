# MacNote - Clone of macOS's Notes Application
 
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

## Firebase related commands

### Serve with firebase locally
```
firebase serve
```
### Deploy
```
firebase deploy
```
